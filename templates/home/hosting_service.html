Servicio de creación de Virtual Hosts

Descripción

Este servicio ofrece mejorar el acceso a una solución web, transformando un URL largo a uno más corto y descriptivo. Esto con el fin de facilitar el acceso a la página y hacerla más memorable.

Clasificación del Servicio

SOFTWARE Y HARDWARE

Disponibilidad

Este servicio está disponible 7 días a la semana durante las 24 horas del día.

Disponibilidad Meta

Se intenta que la disponibilidad meta de este servicio sea de un 99.99%.

Respaldo

Respaldos completos semanales y respaldos incrementales diarios, dependiendo del tipo de solución web.

Representantes del Servicio

Equipo de desarrollo web.

Importancia

Este servicio se considera de tipo complementario, ya que no es completamente necesario para el funcionamiento de la operación como tal, sin embargo hace a la solución web más atractiva para el cliente al facilitar su acceso por medio de una URL más legible y fácil de recordar..
